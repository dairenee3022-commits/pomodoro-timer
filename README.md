# 番茄钟 Pomodoro Timer

一个支持 8 种视觉风格和双向视频通话的网页版番茄钟工具。

## 功能

- **番茄钟循环**：25 分钟专注 → 5 分钟短休息 → 每 4 轮一次 25 分钟长休息
- **8 种视觉风格**：极简主义 / 赛博朋克 / 电子风格 / 液晶风格 / 自然有机 / 可爱风格 / 明星风格 / 异地情侣
- **环形进度条 + 数字时钟**：倒计时进度一目了然
- **可爱角色壁纸**：8 个卡通角色的全屏壁纸，可手动调整位置
- **明星壁纸**：8 位明星的全屏壁纸，可手动调整位置
- **异地情侣双向视频**：Canvas 帧中继 + WebSocket 信令，同一局域网内两台设备面对面视频通话
- **激励语**：每种风格专属的专注激励文案
- **声音提醒**：计时结束时合成提示音
- **历史记录**：浏览器 localStorage 存储，最多 100 条
- **响应式布局**：适配电脑、平板、手机
- **键盘快捷键**：空格 = 开始/暂停，R = 重置，S = 跳过

## 使用方式

### 番茄钟

直接双击打开 `index.html` 即可使用番茄钟功能。

### 双向视频通话（异地情侣模式）

视频通话需要启动本地 HTTPS 服务器（浏览器要求安全上下文才能访问摄像头）：

```bash
npm install
npm start
```

1. 桌面端打开 `https://localhost:3456`
2. 手机端（同一 WiFi）打开 `https://<你的局域网IP>:3456`
3. 首次打开需跳过自签名证书警告（高级 → 继续前往）
4. 两台设备都切换到「异地情侣」风格
5. 都点击「打开摄像头」授权
6. 输入相同的房间号，一台点「创建房间」，另一台点「加入房间」

## 技术方案

- **番茄钟核心**：纯 HTML/CSS/JS 单文件，零后端依赖
- **视频通话**：Canvas 帧中继（JPEG 压缩）+ WebSocket 信令 + Node.js HTTPS 服务器
- **存储**：localStorage（番茄钟数据 + 壁纸位置缓存）
- **无需外网**：视频通话在局域网内 P2P 传输，不依赖任何第三方服务

## 如何替换图片

打开 `index.html`，找到 `cuteCharacters` 或 `celebrities` 数组，修改 `imgUrl` 字段为你的图片路径即可。
